# Quizmas Assets

This folder contains assets for the Quizmas application.

## Structure

- `/sounds/` - Audio files for game effects
- `/images/` - Image assets
- `favicon.ico` - Site favicon

## Sound Files Needed

Create or download the following MP3 files:
- `correct.mp3` - Played when answer is correct
- `wrong.mp3` - Played when answer is wrong
- `tick.mp3` - Timer tick sound
- `countdown.mp3` - Countdown sound
- `winner.mp3` - Victory fanfare
- `join.mp3` - Player joined sound
- `timesup.mp3` - Time's up sound
- `reveal.mp3` - Answer reveal sound
- `background.mp3` - Optional background music

## Free Sound Resources

You can find free sounds at:
- https://freesound.org
- https://soundbible.com
- https://mixkit.co/free-sound-effects/

## Placeholder Image

Create a `quiz-placeholder.png` (400x300px) for quizzes without cover images.
